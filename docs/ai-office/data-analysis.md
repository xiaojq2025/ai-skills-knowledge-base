# AI 数据分析：自然语言驱动 Excel/CSV

> 不会写 Excel 公式？不懂 Python？用自然语言描述需求，AI 帮你分析数据、生成图表、撰写洞察。

---

## 🎯 核心工作流

```
数据文件 + 你的问题 → AI 分析 → 结果 + 图表 + 洞察
```

**适用场景**：
- 销售数据分析（趋势、排名、异常）
- 财务报表整理（汇总、对比、比率）
- 用户行为分析（留存、转化、漏斗）
- 考勤/绩效统计
- 市场调研数据整理

---

## 📝 数据分析 Prompt 模板

### 基础版

```
我有一份数据，请帮我分析：

【数据描述】
- 文件类型：{Excel/CSV/表格}
- 数据内容：{是什么数据，每列代表什么}
- 数据量：{大约多少行}

【我的问题】
1. {具体问题1}
2. {具体问题2}

【输出要求】
- 分析结果用表格呈现
- 关键发现用 bullet points 总结
- 如有异常数据，标红并解释可能原因
```

### 完整示例

```
我有一份销售数据，请帮我分析：

【数据描述】
- 文件类型：Excel
- 数据内容：2024年1-6月各区域销售额、销售人员、产品类别
- 列包括：日期、区域、销售员、产品、金额、数量
- 数据量：约 2000 行

【我的问题】
1. 哪个区域销售额最高？增长率如何？
2. Top 10 销售员是谁？
3. 哪类产品表现最好？
4. 有没有异常数据（如单笔金额过大/过小）？
5. 帮我预测下季度趋势

【输出要求】
- 每个问题单独回答
- 用 Markdown 表格呈现排名和对比
- 关键数字加粗
- 给出 3 条可执行的业务建议
```

## 📊 常见分析场景

### 1. 销售/业绩分析

```
请分析这份销售数据：
- 计算各区域/各月/各产品的销售额、同比、环比
- 找出增长最快和最慢的区域，分析可能原因
- 识别 Top 20% 客户贡献了多少收入（帕累托分析）
- 输出一个执行摘要（给领导看的，200字以内）
```

### 2. 财务分析

```
请分析这份财务报表：
- 计算关键比率：毛利率、净利率、费用率
- 对比预算和实际，找出偏差最大的 3 个项目
- 现金流趋势分析
- 用通俗语言解释给非财务背景的领导
```

### 3. 用户/客户分析

```
请分析这份用户数据：
- 用户分层：按消费金额分高/中/低价值用户
- 计算留存率（次日、7日、30日）
- 找出流失预警用户（多久没活跃）
- 给运营团队 3 条具体建议
```

### 4. 考勤/人力分析

```
请分析这份考勤数据：
- 统计各部门出勤率、迟到率、加班时长
- 找出异常模式（如某人频繁迟到）
- 计算人均工时和效率指标
- 输出给 HR 的月度报告框架
```

## 📈 让 AI 生成图表

如果 AI 支持代码执行（如 Claude Artifacts、ChatGPT Code Interpreter），可以让它生成可视化图表：

```
基于以上分析，请生成以下图表（用 Python matplotlib）：
1. 月度销售额趋势折线图
2. 各区域销售额占比饼图
3. 产品类别对比柱状图

要求：
- 配色专业，适合放入 PPT
- 添加数据标签
- 图例清晰
```

**如果不支持代码执行**，让 AI 输出图表描述，你用 Excel/在线工具制作：

```
请描述一个适合放入 PPT 的图表：
- 图表类型
- X轴/Y轴内容
- 关键数据点
- 配色建议
- 标题和注释
```

## 🛠️ 实战练习

**练习**：找一份你手头的真实数据（Excel/CSV），用 AI 分析。

**检查清单**：
- [ ] 数据已脱敏（去除姓名、手机号等隐私信息）

- [ ] 提出了明确的问题
- [ ] AI 的结果经过了事实核对
- [ ] 洞察部分有业务价值，不是纯数字罗列

## 💡 高级技巧

### 技巧 1：让 AI 先理解数据结构
> "在分析之前，请先总结这份数据的结构：有多少列、每列是什么类型、有没有缺失值、有没有异常值。"

### 技巧 2：对比分析
> "这是本月数据和上月数据，请对比分析变化，并用'上升/下降/持平'标注每个指标。"

### 技巧 3：生成报告大纲
> "基于以上分析，帮我写一份给总经理的数据分析报告大纲，包含：背景、方法、关键发现、建议、附录"

### 技巧 4：数据清洗
> "这份数据有很多格式问题，请帮我：
> 1. 统一日期格式
> 2. 删除重复行
> 3. 填充缺失值（用合理的方式）
> 4. 标准化单位"

## ⚠️ 数据安全提醒

| 场景 | 处理方式 |
|------|---------|
| 公开数据/脱敏数据 | 可直接用云端 AI |
| 含个人隐私（姓名、电话、地址） | 先脱敏或替换为代号 |
| 公司机密财务数据 | 用本地模型或公司授权的 AI 工具 |
| 客户交易明细 | 必须经过合规审批 |
