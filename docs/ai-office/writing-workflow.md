# 文案与报告撰写：AI 辅助长文本生产

> 周报、月报、项目总结、活动方案、标书——用 AI 把"从零写"变成"改草稿"。

---

## 🎯 核心工作流

```
素材收集 → AI 生成初稿 → 人工精修 → 排版输出
 (你)       (AI)          (你)       (AI辅助)
```

**核心原则**：你给 AI "素材和方向"，AI 给你 "结构和草稿"，你负责 "判断和润色"。

---

## 📝 通用长文本 Prompt 模板

```
请帮我撰写一份{文档类型}：

【基本信息】
- 标题/主题：{}
- 目标读者：{}
- 用途：{内部汇报/对外发布/向上申请}
- 字数要求：{}

【必须包含的内容】
1. {要点1}
2. {要点2}
3. {要点3}

【参考素材】
{粘贴相关数据、背景信息、过往文档}

【风格要求】
- {正式/活泼/简洁/详细}
- 多用数据/案例/少空话
- 适合{打印/PPT演讲/邮件发送}

【结构要求】
{指定章节，或让 AI 先出大纲你确认}
```

## 📄 5 大常见文档类型

### 1. 周报/月报

```
请帮我写本周工作总结：

【我的本周工作】
{列出你做的事，哪怕是零散的任务}

【数据/成果】
{任何可以量化的结果}

【下周计划】
{已确定要做的事情}

【需要协调/支持】
{需要领导或其他部门帮忙的}

【要求】
- 用"结论先行"结构
- 每项工作包含：做了什么 + 结果如何
- 困难不回避，但要附带你的应对
- 总字数 300-500 字
```

**AI 输出后，你检查**：
- [ ] 有没有夸大数据？
- [ ] 困难描述是否客观？
- [ ] 有没有遗漏重要工作？
- [ ] 语气是否合适（不卑不亢）？

---

### 2. 项目方案/策划书

```
请帮我写一份项目方案：

【项目】：{名称}
【背景】：{为什么要做}
【目标】：{具体、可衡量}
【范围】：{做什么、不做什么}
【资源】：{预算、人力、时间}

【要求】
- 包含：背景、目标、执行计划、风险、预算、预期收益
- 执行计划按时间线排列
- 风险要有应对措施
- 给领导看的，要有说服力
```

**技巧**：让 AI 先出大纲，你确认后再逐章写。

---

### 3. 活动/会议通知

```
请帮我写一份活动通知：

【活动】：{名称}
【时间】：{}
【地点/线上链接】：{}
【对象】：{谁需要参加}
【目的】：{为什么要参加}

【要求】
- 开头一句话说明价值（为什么要来）
- 时间地点醒目
- 需要准备什么
- 报名/回复方式
- 语气{正式/轻松/活泼}
```

---

### 4. 工作总结/述职报告

```
请帮我写一份年度工作总结：

【岗位】：{}
【时间范围】：{}
【主要工作】：{列出 3-5 项核心工作}
【关键成果】：{数据、奖项、突破}
【不足/反思】：{诚恳但不过度自责}
【明年计划】：{3-5 项}

【要求】
- STAR 法则描述重点工作
- 成果用数据说话
- 不足要有改进措施
- 计划要具体可衡量
- 适合述职演讲，每部分 1-2 分钟
```

---

### 5. 标书/投标文件（片段）

```
请帮我撰写投标文件中的"公司介绍"部分：

【公司信息】
{成立时间、规模、主营业务、核心优势}

【招标项目】
{项目名称、招标方需求}

【要求】
- 突出与招标需求的匹配度
- 用案例证明能力
- 语言正式、严谨
- 控制在一页 A4 纸
```

## 🛠️ 长文本进阶技巧

### 技巧 1：先大纲，后内容
> "请先给我一份大纲，包含每个部分的标题和 1 句话说明。我确认后，你再逐部分展开。"

### 技巧 2：分段迭代
> "我们先写第一部分'背景'。写完后停一下，我确认方向后再继续。"

### 技巧 3：角色扮演
> "你是一位有 20 年经验的咨询公司顾问，请用专业术语和结构化思维帮我写这份方案。"

### 技巧 4：对比改写
> "这是我写的初稿，请你优化：
> 1. 让逻辑更清晰
> 2. 删除重复和冗余
> 3. 增加过渡句让段落衔接更自然
> 4. 保持原有数据和事实不变"

### 技巧 5：多版本输出
> "请写两个版本：
> - 详细版：800字，适合正式提交
> - 精简版：200字，适合口头汇报"

## ⚡ 快速检查清单

AI 生成文档后，按这个清单检查：

```markdown
[ ] 事实准确：数据、人名、时间、地点都没错
[ ] 逻辑通顺：因果关系成立，不跳跃
[ ] 无 AI 幻觉：没有 AI 自己编造的内容
[ ] 语气合适：符合公司文化和读者身份
[ ] 格式规范：标题层级、标点、编号统一
[ ] 长度合适：符合要求，不啰嗦不欠缺
[ ] 敏感信息：没有泄露机密或隐私
```
