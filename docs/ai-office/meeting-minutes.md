# AI 会议纪要：从录音到行动清单

> 会议 1 小时，整理纪要 30 分钟？用 AI 把录音/速记转成结构化纪要，自动提取决策项和待办事项。

---

## 🎯 核心工作流

```
录音/速记 → 转文字（如有必要）→ AI 提取 → 人工校对 → 分发
```

**工具组合**：
- 录音：手机自带录音、飞书妙记、通义听悟
- 转文字：飞书妙记（免费）、讯飞听见、剪映（免费）
- 整理：**AI（就是你正在用的 Kimi/Claude）**

---

## 📝 会议纪要 Prompt 模板

### 基础版（有文字稿时使用）

```
请帮我把以下会议记录整理成标准会议纪要：

【会议要求】
- 提取所有"决策结论"（用 ✅ 标记）
- 提取所有"待办事项"（用 ☐ 标记，标注负责人和截止日期）
- 提取所有"有争议未决"的事项（用 ❓ 标记）
- 删除闲聊、重复、与议题无关的内容
- 保留关键数据、数字、时间节点

【输出格式】
会议主题：
时间：
地点/线上：
参会人：
缺席人：

一、会议议程
（列出讨论的议题）

二、决策结论
（用 ✅ 标记）

三、待办事项
（用 ☐ 标记，负责人+截止日期）

四、争议事项
（用 ❓ 标记，说明分歧点）

五、其他备忘

---
【会议记录原文如下】
[粘贴文字稿]
```

## 📋 进阶版：针对不同会议类型的优化

### 类型 1：周会/例会

```
【额外要求】
- 与上周待办对比：哪些完成了？哪些延期？原因是什么？
- 本周重点工作（TOP 3）
- 需要协调的资源/风险
```

### 类型 2：项目评审会

```
【额外要求】
- 项目当前状态：绿/黄/红灯？
- 里程碑完成情况
- 风险项及应对措施
- 是否需要调整计划？
```

### 类型 3：决策会/头脑风暴

```
【额外要求】
- 每个方案的优点、缺点、风险
- 最终选择的方案及理由
- 反对意见及如何回应
```

### 类型 4：客户/外部会议

```
【额外要求】
- 客户提出的需求和痛点
- 我方承诺的事项（需特别注意）
- 下一步跟进计划
- 客户情绪/态度变化（如能从文字中判断）
```

## 🛠️ 实战案例

**输入（原始速记）**：
```
小王：上周用户增长数据出来了，新增 1200，比目标少了 300
小李：我觉得是投放渠道的问题，抖音那边 ROI  dropped
小王：对，抖音上周调整了算法，我们得换策略
张总：那下个月重点转小红书？
小李：可以，但小红书需要新素材，设计那边排满了
张总：那加急吧，本周内出 5 套素材，小李负责跟进
小王：技术这边有个问题，APP 闪退率上升到 3%，需要修
张总：优先级最高，小王你牵头，周三前给我方案
小王：好。另外下个月活动预算...
张总：预算按原计划，不超 10 万
```

**AI 输出**：

```markdown
# 会议纪要

**主题**：周会：用户增长与产品问题
**时间**：202X-XX-XX
**参会人**：小王、小李、张总

## 一、决策结论 ✅
1. ✅ 下个月投放策略从抖音转向小红书
2. ✅ 本月活动预算上限 10 万元
3. ✅ APP 闪退修复优先级最高

## 二、待办事项 ☐
| 事项 | 负责人 | 截止日期 |
|------|--------|---------|
| ☐ 产出小红书素材 5 套 | 小李 | 本周内 |
| ☐ 提交 APP 闪退修复方案 | 小王 | 周三前 |
| ☐ 跟进素材制作进度 | 小李 | 持续 |

## 三、争议事项 ❓
- 无

## 四、关键数据
- 上周新增用户：1200（目标 1500，缺口 300）
- APP 闪退率：3%（需降低）
- 活动预算上限：10 万元

## 五、风险备忘
- 抖音算法调整导致 ROI 下降，需持续观察
- 设计团队资源紧张，可能影响素材产出速度
```

## 💡 提效技巧

### 技巧 1：让 AI 对比多期纪要
> "以下是上上周和上周的会议纪要，请对比：
> 1. 哪些待办完成了？
> 2. 哪些延期了？
> 3. 重复出现的问题是什么？"

### 技巧 2：自动生成待办追踪表
> "基于以上会议纪要，生成一个 Excel/表格格式的待办追踪表，包含：事项、负责人、截止日期、当前状态"

### 技巧 3：生成会议通知模板
> "基于以上纪要，写一封给未参会人员的'会议通报'邮件，概括决策和与他们相关的待办"

### 技巧 4：语音速记直接丢给 AI
如果你用飞书妙记/讯飞生成文字稿，直接全文粘贴，AI 会自动过滤语气词（"嗯"、"啊"、"那个"）和重复内容。

---

## ⚠️ 注意事项

1. **隐私**：涉及敏感信息（薪资、人事、商业机密）的会议，不要用云端 AI 处理。用本地部署模型或脱敏后处理。
2. **准确性**：AI 可能误解专业术语或人名，必须人工校对。
3. **责任**：会议纪要发送前，让相关负责人确认待办事项和截止日期是否准确。
