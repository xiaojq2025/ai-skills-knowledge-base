# AI 辅助邮件撰写：从草稿到发送

> 商务邮件、内部沟通、请示报告、催办函——用 AI 在 1 分钟内生成专业得体版本，你来把关后发送。

---

## 🎯 核心工作流

```
你提供要点 → AI 生成草稿 → 你修改发送
  (30秒)      (30秒)       (30秒)
```

**关键**：AI 写 80%，你改 20%。你负责把关事实准确性和语气是否合适。

---

## 📝 通用邮件 Prompt 模板

### 基础版（适合大多数场景）

```
请帮我写一封商务邮件：

【收件人】：{姓名/职位}
【目的】：{你想让对方做什么}
【背景】：{相关上下文}
【语气】：{正式/友好/紧迫/委婉}
【长度】：{简短（100字内）/标准（200字内）/详细}
【必须包含的信息】：{列出所有必须提到的点}
【禁忌】：{不要说什么/不要假设什么}

输出格式：
主题：
正文：
```

### 完整示例

```
请帮我写一封商务邮件：

【收件人】：采购部张经理
【目的】：请求加快审批我部门的设备采购申请
【背景】：申请已提交两周，项目下周启动，设备未到会影响进度
【语气】：礼貌但略带紧迫
【长度】：标准（200字内）
【必须包含的信息】：申请单号2024-056、设备名称、希望本周五前批复
【禁忌】：不要指责对方慢、不要提到领导施压

输出格式：
主题：
正文：
```

## 📧 6 大常见邮件场景

### 1. 会议邀请/通知

```
角色：行政助理
任务：发送会议通知
背景：{会议主题、时间、地点/链接、参会人员}
要求：
- 列出议程（3-4项）
- 说明需要提前准备的材料
- 提醒准时参加
格式：邮件，包含日历附件提示
```

### 2. 会议延期/取消

```
角色：行政经理
任务：通知会议延期
背景：原定{时间}的{会议名称}因{原因}需调整
要求：
- 表达歉意
- 给出 2-3 个备选时间请大家投票/确认
- 说明已确定的事项不受影响
语气：诚恳、专业
```

### 3. 工作汇报/请示

```
角色：{你的职位}
任务：向{领导}汇报{项目}进展并请示下一步
背景：{目前完成情况、遇到的问题}
要求：
- 用"结论先行"结构：先说结果，再说过程
- 问题要附带建议方案（A/B选项）
- 明确你需要领导决策什么
格式：分点清晰，每点不超过2行
```

### 4. 催办/跟进

```
角色：项目经理
任务：跟进{事项}进度
背景：{事项}原定{日期}完成，目前尚未收到反馈
要求：
- 先肯定对方之前的工作
- 温和地询问当前状态
- 给出新的 deadline（留有余地）
- 表达愿意提供支持
禁忌：不要用"尽快"、"赶紧"等压迫性词汇
```

### 5. 跨部门协作请求

```
角色：{你的职位}
任务：请求{部门}协助完成{事项}
背景：{为什么需要他们、项目整体目标}
要求：
- 说明这件事对双方的价值
- 明确需要对方做什么、截止时间
- 提供对方需要的背景材料
语气：平等协作，不是上级命令
```

### 6. 道歉/解释

```
角色：{你的职位}
任务：就{失误/延迟}向{对象}道歉并说明情况
背景：{发生了什么、为什么会发生}
要求：
- 不找借口，真诚道歉
- 简要说明原因（不推卸责任）
- 给出补救措施和预防方案
- 请求对方谅解
语气：诚恳、负责
```

## ✉️ 邮件结构黄金模板

AI 生成邮件时，要求它遵循这个结构：

```
【主题行】简明扼要，包含关键词
  例："关于Q3营销方案的意见征求（周五前反馈）"

【称呼】根据关系选择
  正式："尊敬的王总："
  内部："张经理，您好！"
  熟悉："李姐，好！"

【开场】一句话说明来意
  "冒昧打扰，是想就XX事项向您请示。"

【正文】分段清晰，每段一个主题
  - 背景（为什么写这封邮件）
  - 具体事项（是什么）
  - 行动要求（需要对方做什么）

【结尾】明确下一步
  "期待您的回复，如有疑问请随时联系我。"

【落款】
  姓名
  职位
  联系方式
  日期
```

## 🛠️ 实战练习

**练习**：用 AI 写一封你本周真实需要发的邮件。

1. 从上面选一个最贴近的场景模板
2. 填入你的真实信息
3. AI 生成后，检查：
   - [ ] 事实是否准确？
   - [ ] 语气是否合适？
   - [ ] 是否遗漏了必须说的信息？
   - [ ] 有没有 AI 自己脑补的不实内容？
4. 修改后发送

## 💡 高级技巧

### 让 AI 写多个版本供你选择
> "请写两个版本：A版语气委婉适合对方心情可能不好时；B版简洁直接适合对方很忙时。"

### 让 AI 帮你改你自己写的草稿
> "这是我写的草稿，请你优化：
> 1. 让语气更专业
> 2. 缩短到150字以内
> 3. 确保没有语法错误
> 
> [粘贴你的草稿]"

### 让 AI 生成回复建议
> "我收到了这封邮件，请帮我写 3 个不同立场的回复草稿：
> 1. 接受对方条件
> 2. 委婉拒绝并给出替代方案
> 3. 请求延期回复
>
> [粘贴对方邮件]"
