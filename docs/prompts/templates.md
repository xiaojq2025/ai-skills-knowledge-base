# Prompt 模板库：即开即用

> 办公场景最常用的 Prompt 模板，复制粘贴即可使用。按场景分类，支持快速微调。

---

## ✉️ 邮件类

### 会议邀请
```
请写一封会议邀请邮件：
- 会议主题：{}
- 时间：{}
- 地点/线上链接：{}
- 参会人员：{}
- 议程：{1. ... 2. ... 3. ...}
- 需要提前准备：{}
- 语气：{正式/轻松}
```

### 催办进度
```
请写一封跟进邮件：
- 跟进事项：{}
- 原定完成时间：{}
- 收件人：{}
- 关系：{同事/下属/合作方}
- 语气：{礼貌但紧迫/友好提醒}
- 要求：先肯定对方工作，再询问状态，给新 deadline
```

### 道歉/解释
```
请写一封道歉邮件：
- 事由：{}
- 原因：{}
- 影响：{}
- 补救措施：{}
- 收件人：{}
- 要求：真诚不找借口，说明原因不推卸责任，给出补救方案
```

## 📄 文档类

### 周报生成
```
请帮我写本周工作总结：
本周完成工作：{列出事项}
关键数据/成果：{}
遇到的问题：{}
下周计划：{}
需要支持：{}
要求：结论先行，数据说话，300-500字
```

### 会议纪要整理
```
请将以下会议记录整理成标准纪要：
要求：
- 提取决策（✅）、待办（☐ 负责人+时间）、争议（❓）
- 删除闲聊和重复
- 保留关键数据
原文：{粘贴记录}
```

### 方案大纲
```
请为"{}"项目写一份方案大纲，包含：
1. 背景与问题
2. 目标（SMART）
3. 执行计划（时间线）
4. 资源需求
5. 风险与应对
6. 预期收益
要求：每个部分只写标题+2-3句话说明，不展开
```

## 📊 数据分析类

### 销售分析
```
请分析以下销售数据：
数据说明：{列名说明}
问题：
1. 各区域/产品销售情况对比
2. 趋势分析（增长/下降）
3. 异常数据识别
4. Top/Bottom 排名
5. 业务建议
数据：{粘贴表格/CSV}
```

### 数据清洗
```
请帮我清洗这份数据：
问题：
- 日期格式不统一
- 有空值/重复值
- 单位不一致
- 文本格式混乱
要求：
- 输出清洗后的数据
- 说明每处修改及原因
数据：{粘贴}
```

## 💬 沟通类

### 向上汇报
```
请帮我准备向{领导}汇报的内容：
汇报主题：{}
背景：{}
关键信息：{}
我的诉求：{需要决策/需要资源/需要支持}
要求：
- 3分钟说完
- 先说结论，再说理由
- 给选项而不是问答题
```

### 跨部门协作请求
```
请帮我写一段跨部门协作的请求：
目标部门：{}
协作事项：{}
对对方的价值：{}
需要对方做什么：{}
截止时间：{}
语气：平等协作，不是命令
```

### 拒绝请求（委婉）
```
请帮我写一段委婉拒绝的话：
对方请求：{}
拒绝原因：{}
替代方案：{}
关系：{同事/合作方/客户}
要求：
- 表达理解和感谢
- 说明困难（客观）
- 给出替代方案
- 保持关系友好
```

## 🎨 创意类

### 标题生成
```
请为以下内容生成 10 个标题：
内容主题：{}
风格：{正式/吸睛/简洁/专业}
要求：
- 适合{邮件主题/文章标题/PPT标题}
- 10个不同角度
- 标出你推荐的 Top 3
```

### 文案改写
```
请改写以下文案：
原文：{}
要求：
- 目标受众：{}
- 语气：{}
- 长度：{}
- 突出：{核心卖点}
输出 3 个版本供选择
```

## 🧠 思考类

### 问题拆解
```
我面临一个问题：{}
请帮我拆解：
1. 这个问题的本质是什么？
2. 有哪些可能的解决方向？
3. 每个方向的优缺点？
4. 推荐哪个方向？为什么？
```

### 决策分析
```
我需要做一个决定：{}
选项：
A. {}
B. {}
C. {}

请帮我分析：
- 每个选项的利弊
- 风险点
- 需要什么资源
- 如果选错了怎么补救
- 你的推荐
```

---

## ⚙️ 使用说明

1. **复制模板**
2. **替换 `{}` 里的内容**
3. **根据实际需求增删要求**
4. **AI 输出后检查事实准确性**
5. **修改后使用**
